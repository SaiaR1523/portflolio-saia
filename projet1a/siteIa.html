Bienvenur sur mon premier projet IA
Le concept est le suivant :
A l'aide de la détection de main proposé par Mediapipe,
on enregistre une position donnée
Une fois cette position stocké dans une variable elle permet alors de changer la couleur de l'arrière plan
Ainsi les positions retenues sont les suivantes : (et les couleurs attendus)
    Toutes les doigts écartés : vert
    Indexe levé : jaune
    Majeur en hautpouce levé : rose
    Indexe et majeur levé : rouge
    Chifre trois avec l'indexe, majeure, annulaire : bleu

Labo, avec Hands de mediapipe

Attribution de position d'une main, pour changer l'arrière plan
